IMAGES FOLDER - Inventory Management Web App
=============================================

Place your spare part / product images here.

Folder structure:
  images/
  └── products/     ← Put all product/spare part photos here

Naming tip:
  Name each image after its Product Code for easy reference.
  Examples:
    PRD-001.jpg   → Wireless Keyboard
    PRD-002.jpg   → USB-C Hub 7-Port
    PRD-003.jpg   → Ergonomic Office Chair

Supported formats: .jpg, .jpeg, .png, .webp, .gif

To display an image in the web app, reference it like this in index.html:
  <img src="images/products/PRD-001.jpg" alt="Wireless Keyboard" />
